# 💳 Alke Wallet

Aplicación frontend de billetera digital desarrollada con HTML, CSS y JavaScript.

## 📌 Funcionalidades

- Inicio de sesión
- Menú principal
- Depósito de dinero
- Envío de dinero
- Historial de transacciones

## 🛠️ Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## 📁 Estructura del proyecto

- index.html → login
- menu.html → menú principal
- deposit.html → depósito
- sendmoney.html → envío
- transactions.html → historial
- js/script.js → lógica
